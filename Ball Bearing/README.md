# Ball Bearing Assembly

## Overview

This project is a 3D model and technical drawing of a **ball bearing assembly** designed in Autodesk Inventor.

The bearing is used to reduce friction between rotating components and support radial loads while allowing smooth rotational motion.

The project includes:
- a fully assembled 3D model
- an exploded (presentation) view
- a detailed technical drawing with dimensions and parts list

---

## System Description

The ball bearing consists of the following main components:

- **Outer Ring** – stationary outer structure
- **Inner Ring** – rotating inner element
- **Balls (rolling elements)** – transmit load and reduce friction
- **Cage** – maintains spacing and alignment of balls

According to the assembly drawing:
- 1 × outer ring  
- 1 × inner ring  
- 8 × balls  
- 2 × cage elements :contentReference[oaicite:0]{index=0}  

All components are modeled using stainless steel.

---

## How It Works

The bearing operates by converting sliding friction into rolling friction:

- The **inner ring** rotates relative to the outer ring
- The **balls** roll between the two rings
- The **cage** keeps the balls evenly spaced and prevents contact between them

This design:
- minimizes friction
- reduces wear
- allows smooth and efficient rotation

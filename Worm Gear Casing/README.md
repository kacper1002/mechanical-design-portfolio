# Worm Gear Housing

##  Overview

This project is a 3D model and technical drawing of a **worm gear housing** designed in Autodesk Inventor.

The housing is intended to enclose and support a worm gear transmission system, providing:
- structural support
- precise alignment of components
- protection of internal mechanisms

The project includes:
- a fully modeled housing geometry
- multiple section views
- a detailed technical drawing with dimensions and machining features

---

## System Description

The worm gear housing is designed to accommodate a worm gear assembly, which typically consists of:
- a **worm shaft**
- a **worm wheel**
- supporting bearings and shafts

The housing ensures:
- proper positioning of shafts
- alignment between worm and gear
- structural rigidity under load

The design includes key features such as:
- bearing seats for shaft support
- mounting interfaces
- internal cavities for gear operation
- access points for assembly and maintenance

---

##  Function

Worm gear systems are used for:
- **high torque transmission**
- **speed reduction**
- **compact power transfer**

The housing plays a critical role by:
- maintaining correct gear meshing geometry
- absorbing mechanical loads
- protecting internal components from contamination
